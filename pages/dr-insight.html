Project Overview

Our AI-powered system leverages cutting-edge deep learning and machine learning techniques to analyze retina images, classify the severity of Diabetic Retinopathy, and assist healthcare providers in making informed decisions. It is capable of both binary (presence/absence) and multiclass (severity levels) classifications.

Features

  • Classification of retina images into binary (No DR / DR) and multiclass (No DR, Mild NPDR, Moderate NPDR, Severe NPDR, PDR) categories.
  • Machine Learning Models (ML): Utilized HOG (Histogram of Oriented Gradients) features and trained seven different models.
  • Deep Learning Models (DL): CNN-based architectures (MobileNetV2, InceptionV3) for automatic feature extraction and classification.
  • Hybrid Models (DL + ML): Combined deep learning feature extraction with ML classifiers to enhance accuracy.
  • Comprehensive analysis with:
    • Binary classification accuracy up to 96.59%.
    • Multiclass classification accuracy up to 77.35%.
    • Feature extraction techniques improving interpretability.
  • Selected Model for Frontend:
    • Binary Classification: MobileNetV2 + MLP
    • Multiclass Classification: MobileNetV2 + SVM
    • Feature extraction techniques improving interpretability.
  • User-friendly interface with explanations of the disease, detection methods, and severity classification.
  • Firebase-connected contact system for user inquiries and feedback

Technical Implementation

The application uses Flask for the backend, with a PostgreSQL database to store users queries. The feature extraction is implemented using TensorFlow and machine learning models are implemented using scikit-learn and are integrated into the Flask application using joblib for model serialization.

Model Performance Comparison

  • Deep Learning Models:
    • InceptionV3: 96.32% binary, 76.67% multiclass accuracy
    • MobileNetV2: 95.36% binary, 77.35% multiclass accuracy
  • Hybrid Approach (Best Performance):
    • MobileNetV2 + MLP: 96.59% binary classification
    • MobileNetV2 + SVM: 77.35% multiclass classification

Diabetic Retinopathy Stages

The research identified five critical stages of Diabetic Retinopathy:

  1. No DR (Healthy Retina)
  2. Mild Non-Proliferative DR (NPDR)
  3. Moderate NPDR
  4. Severe NPDR
  5. Proliferative DR (PDR)
